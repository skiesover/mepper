## Installation

1. Install Node.JS
2. Clone project to your computer
3. In Terminal go to your project's root folder:
`cd ../mepper/`
4. install dependencies: `npm install --save-dev webpack webpack-dev-server webpack-cli`
5. run `npm start`
6. In browser go to `http://localhost:8080/`
